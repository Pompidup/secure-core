## App Title

**SecureCore** (10 chars)

## Short Description (< 80 chars)

Coffre personnel de documents chiffres, 100% hors-ligne.

## Full Description

SecureCore est un coffre-fort numerique pour vos documents sensibles. Vos fichiers sont chiffres localement sur votre appareil et n'en sortent jamais.

### Fonctionnalites

- Chiffrement AES-256-GCM de chaque document avec une cle unique
- Cles gerees par le hardware Android Keystore
- Deverrouillage biometrique (empreinte) ou code appareil
- Support des images (JPEG, PNG, WebP), PDF et fichiers texte
- Previsualisation securisee avec purge automatique
- Aucune connexion internet requise

### Garanties de securite

- Vos documents ne quittent JAMAIS votre appareil
- Aucune donnee n'est envoyee a des serveurs externes
- Aucune collecte de donnees, aucun tracking, aucune telemetrie
- Le backup Android est desactive : vos documents ne vont pas dans Google Drive
- Chaque document est chiffre avec sa propre cle de chiffrement
- Les fichiers temporaires de previsualisation sont purges automatiquement

### Comment ca marche

1. Importez un document depuis votre appareil
2. SecureCore le chiffre immediatement et supprime la copie en clair
3. Pour consulter un document, authentifiez-vous par biometrie
4. Le document est dechiffre temporairement puis purge de la memoire

### Limites de la V1

- Android uniquement (iOS prevu pour une version ulterieure)
- Taille maximale par document : 50 Mo
- Types supportes : JPEG, PNG, WebP, PDF, texte brut
- Si vous reinstallez l'application, vos documents sont perdus definitivement (les cles de chiffrement sont liees a l'installation)
- La recuperation de donnees apres reinstallation est prevue pour la V2

### Promesse

Vos documents sont a vous. Ils restent sur votre appareil, chiffres, et sous votre controle. Personne d'autre — ni nous, ni Google, ni un attaquant — ne peut y acceder.
